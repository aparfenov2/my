требования
--------------------
- программируемая задержка
- прямая запись в порт для настройки
- DMA с указанием области отрисовки
	- только 24 bit


реализация
--------------------
регистры
	- управления : 8
		- start dma
		- start_single
		- rst
		- dc
	- флагов : 8
		- dma complete
		- single_complete
	- data/command : 8
	- значения задержки : 8
	- x1,y1,x2,y2, xmax, ymax : 16
	- buf_start : 32
	- buf_end : 32
